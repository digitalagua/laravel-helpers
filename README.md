# Laravel Helpers

##

Adds a few helpers to laravel.

## Installation

Add this repo to your composer.json file.

```
{
  "require": {
    ...
    "digitalagua/laravel-helpers": "dev-master",
    ...
  },
  ...
  "repositories": [
      {
        "type": "git",
        "url": "https://github.com/digitalagua/laravel-helpers"
      }
    ]
  }
}
```

Run `composer update`
